## Stack

* Meteor 1.3.4.4
* Blaze & Handlebars.js (client)
* jQuery, OAuth2, minimongo, Twitter Bootstrap 3.3.6 & Babeljs ES2015 (client)
* Apache Cordova 6.0.0 (iOS 4.1.0 & Android 5.1.1)
* Node.js 0.10.46 (server)
* MongoDB 3.2 (database)
